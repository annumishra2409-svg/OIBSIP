OIBSIP - Level 1 Task 1: Landing Page

Project Title:

Responsive Landing Page using HTML and CSS

Description:

This project is a modern and responsive Landing Page developed as part of the OASIS INFOBYTE Web Development and Designing Internship (Level 1 - Task 1).

The webpage is designed using HTML5 and CSS3 and includes a clean user interface with sections such as:

- Header and Navigation Bar
- Hero Section
- Features Section
- About Us Section
- Footer

The project demonstrates the fundamentals of front-end web development, including proper layout design, semantic HTML structure, Flexbox, color palettes, and responsive styling.

Features:

*Responsive and modern design
*Clean and semantic HTML structure
*Attractive dark theme with green accent colors
*Flexbox-based layout for proper alignment
*Hover effects and smooth transitions
*Beginner-friendly and easy-to-understand code

Technologies Used:

1)HTML5
2) CSS3
3)Flexbox
4)Responsive Web Design Principles

Learning Outcomes

Through this project, I learned:

- Creating webpage structure using HTML5
- Styling webpages using CSS3
- Using Flexbox for layouts and alignments
- Applying color palettes and typography
- Building responsive and visually appealing landing pages
- Organizing project files for GitHub submission

How to Run the Project

1. Download or clone the repository.
