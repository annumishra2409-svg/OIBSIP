📌 Project Title

Personal Portfolio Website

📖 Description

This is a responsive and modern Personal Portfolio Website developed using HTML5 and CSS3. The website showcases my profile, education, skills, internship experience, projects, and contact information in a clean and professional manner.

The main objective of this project is to demonstrate my front-end web development skills, responsive design techniques, and ability to create a professional portfolio website.

✨ Features

- Responsive and user-friendly design
- Navigation bar with smooth section links
- Hero section with profile image and introduction
- About Me section
- Education details
- Skills section
- Internship Experience section
- Projects showcase
- Contact information section
- Mobile-friendly layout using CSS Media Queries
