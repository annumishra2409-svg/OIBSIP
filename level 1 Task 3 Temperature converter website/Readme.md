OIBSIP - Level 1 Task 3: Temperature Converter Website
                                                                
Project title:
Temperature Converter Website using HTML, CSS, and JavaScript

Description about project:

This project is a responsive and interactive Temperature Converter developed as part of the OASIS INFOBYTE Web Development and Designing Internship (Level 1 - Task 3).

The application allows users to convert temperature values between Celsius (°C), Fahrenheit (°F), and Kelvin (K) with an attractive user interface and smooth user experience.

Features of project:

1)Convert temperature between:
  *Celsius (°C)
  *Fahrenheit (°F)
  *Kelvin (K)
2)Real-time validation for invalid input
3)Swap temperature units with one click
4)Responsive design for mobile and desktop devices
5)Modern Glassmorphism UI Design
6)Animated background effects
7)Smooth transitions and hover effects
8)Displays results with proper temperature symbols

Technologies Used:

- HTML5
- CSS3
- JavaScript (ES6)

How to Run the Project:

1. Download or clone this repository.
2. Open the project folder.
3. Open the "index.html" file in any web browser.
4. Enter a temperature value.
5. Select the source and target units.
6. Click on the Convert button to view the result.

Temperature Conversion Formulas

1) Celsius to Fahrenheit:
  F = (C × 9/5) + 32

2)Celsius to Kelvin:
  K = C + 273.15

3)Fahrenheit to Celsius:
  C = (F − 32) × 5/9

4)Fahrenheit to Kelvin:
  K = (F − 32) × 5/9 + 273.15

5)Kelvin to Celsius:
  C = K − 273.15

6)Kelvin to Fahrenheit:
  F = (K − 273.15) × 9/5 + 32
